# AuroGram - Spiritual Quotes from Sri Aurobindo & The Mother

A mobile-first Instagram-style web app dedicated to sharing the timeless wisdom of Sri Aurobindo and The Mother through beautiful quotes and imagery.

---

## Features

- **Instagram-like Feed**: Scroll through beautifully designed quote cards with background images
- **Author Profiles**: Detailed profiles for Sri Aurobindo and The Mother with their biography, photos, and quotes
- **Categories**: Filter quotes by themes - Yoga, Consciousness, Life, Love, Aspiration, Peace, Transformation
- **Search**: Find quotes by text or browse by category
- **Save Quotes**: Bookmark your favorite quotes for easy access
- **Stories**: Quick access to authors and categories at the top of the feed
- **Double-tap to Like**: Instagram-style interaction with heart animation
- **Mobile-First Design**: Optimized for mobile devices with responsive layout

---

## Tech Stack

- **React** 18.2.0 - UI library
- **React Router** 6.20.0 - Client-side routing with HashRouter
- **Vite** 5.0.0 - Build tool and dev server
- **CSS3** - Custom styles with CSS variables

## Pages

| Route | Description |
|-------|-------------|
| `/feed` | Main feed with all quotes |
| `/profile/:authorId` | Profile page for Sri Aurobindo or The Mother |
| `/search` | Search quotes and browse categories |
| `/saved` | View saved/bookmarked quotes |

## Getting Started

### Prerequisites

- Node.js 18+

```bash

# Install dependencies
```

## About the Authors

### Sri Aurobindo (1872-1950)

Sri Aurobindo was an Indian philosopher, yogi, maharishi, poet, and Indian nationalist. He joined the Indian movement for independence from British colonial rule, and became one of its influential leaders. He later became a spiritual reformer, introducing his visions on human progress and spiritual evolution.

### The Mother (1878-1973)

The Mother, born Mirra Alfassa, was a spiritual guru and collaborator of Sri Aurobindo. She was the spiritual leader of the Sri Aurobindo Ashram and founded Auroville, an international township in Tamil Nadu, India.
